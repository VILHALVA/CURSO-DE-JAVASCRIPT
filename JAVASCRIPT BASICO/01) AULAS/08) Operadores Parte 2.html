OPERADORES RELACIONAIS:

  • [>] : Maior quê
  • [<] : Menor quê
  • [>=] : Maior ou igual
  • [<=] : Menor ou igual
  • [==] : Igual
  • [!=] : Diferente

BOLEANO:

  • 5 > 2 = true
  • 7 < 4 = false
  • 8 >= 8 = false
  • 9 <= 7 = false
  • 5 == 5 = true
  • 4 != 4 = false

IDENTIDADE:

  • 5 == 5 = true
  • 5 == "5" = true
  • 5 === "5" = false
  • 5 === 5 = true

LÓGICOS:

  • [!] = NEGAÇÃO (not/não)
  • [&&] = CONJUÇÃO (and/e)
  • [||] = DISJUNÇÃO (or/ou)
